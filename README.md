# Wanderlust Adventures - Travel Website

A modern, responsive travel agency website template built with HTML5, Bootstrap 5, and CSS3.

## Features

- ✅ Fully responsive design
- ✅ Modern and clean UI
- ✅ Contact form with validation
- ✅ Destination gallery
- ✅ Services section
- ✅ About us section
- ✅ Social media integration
- ✅ Smooth scrolling navigation
- ✅ Bootstrap 5 components

## Pages/Sections

1. **Hero Section** - Eye-catching banner with call-to-action
2. **About Us** - Company information and values
3. **Services** - Travel services offered
4. **Destinations** - Popular travel destinations
5. **Contact Form** - Customer inquiry form
6. **Footer** - Contact info and social links

## Customization

### Business Information
1. Change business name in:
   - Navigation bar (`<title>` and `.navbar-brand`)
   - Footer section
   - Contact information

2. Update contact details:
   - Phone number
   - Email address
   - Physical address
   - Social media links

3. Modify services:
   - Update service titles and descriptions
   - Change service icons (Font Awesome)

4. Update destinations:
   - Replace destination images
   - Update destination names and descriptions
   - Add/remove destinations as needed

### Styling
- Primary color: `#2a9d8f` (teal)
- Secondary color: `#264653` (dark blue)
- Accent color: `#e9c46a` (gold)
- Update in CSS `:root` variables

### Images
Replace Unsplash image URLs with your own travel photos:
- Hero background image
- About section image
- Destination images

## Deployment

### Netlify (Recommended)
1. Sign up at [Netlify](https://netlify.com)
2. Drag and drop the `travel-website` folder
3. Your site will be live instantly!

### GitHub Pages
1. Create a GitHub repository
2. Push this code to the repository
3. Enable GitHub Pages in repository settings

### Custom Domain
1. Purchase a domain (e.g., `yourtravelbusiness.com`)
2. Connect it in Netlify/GitHub Pages settings
3. Update any absolute URLs in the code

## Form Handling

The contact form currently shows an alert on submission. For actual form handling:

### Netlify Forms (Free)
1. Add `netlify` attribute to form: `<form netlify>`
2. Netlify will automatically handle form submissions
3. View submissions in Netlify dashboard

### Formspree (Alternative)
1. Sign up at [Formspree](https://formspree.io)
2. Update form action URL
3. Free tier includes 50 submissions/month

## SEO Optimization

1. Update meta tags in `<head>`
2. Add descriptive alt text for images
3. Use semantic HTML elements
4. Add Open Graph tags for social sharing
5. Create a `sitemap.xml` and `robots.txt`
